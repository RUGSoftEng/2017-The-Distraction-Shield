Mixin: optionsPage

optionsPage

This file contains the core functions of the options page. this has all the local variables, initializes everything javascript related and connects the syncStorage, connectDataToHtml, blacklistTable and HtmlFunctionality to one smoothly running file. Besides the initialization it contains the functions to manipulate the local variables found here

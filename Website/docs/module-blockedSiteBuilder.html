Module: blockedSiteBuilder

This module is a utility used to construct and initialize instances of the BlockedSite class. This module is the only right way of creating a new BlockedSiteItem.

Methods

(inner) createBlockedSiteAndAddToStorage(newUrl)

constructs a new BlockedSite and instantly adds it to the storage as well
Parameters:
Name Type Description
newUrl string the unformatted url of which to construct a BlockedSite

(inner) createNewBlockedSite(newUrl)

Constructs a new blockedSite instance using the given url, and passes it on using a promise The only real right way of creating a new blocked site. For this it uses the BlockedSite and the urlFormatter. It also adds it to the current list of blockedSites.
Parameters:
Name Type Description
newUrl string the unformatted url of which to construct a BlockedSite

Module: interception

This module takes care of all the data that needs to be updated when we are intercepted

Methods

(static) addToInterceptDateList()

Adds the current time+date to the saved time+date list

(static) calcInterceptData(dateList)

This method goes through the interceptDateList and count how many times the user was intercepted last day, last week, last month and the total amount of interceptions.
Parameters:
Name Type Description
dateList Array the total list with all interceptions on the different days

(static) incrementInterceptionCounter(urlAddress)

Receives the url from the parameter, and searches the correct blockedSite item from the blockedsite list. Then the interceptioncounter for this item is incremented by 1. Also the global interceptioncounter is incremented by one.
Parameters:
Name Type Description
urlAddress string the url to be compared with the blockedsite list to find the correct item to be incremented
